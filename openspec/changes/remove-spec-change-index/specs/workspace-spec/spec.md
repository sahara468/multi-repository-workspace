## REMOVED Requirements

### Requirement: Workspace specification management
**Reason**: Spec management is handled externally by OpenSpec tooling. The `mrw spec` command is redundant.
**Migration**: Use OpenSpec directly for specification management.

### Requirement: Specification listing
**Reason**: Part of removed spec management functionality.
**Migration**: Use OpenSpec commands for listing specifications.

### Requirement: Specification display
**Reason**: Part of removed spec management functionality.
**Migration**: Use OpenSpec commands for viewing specifications.

### Requirement: Specification creation
**Reason**: Part of removed spec management functionality.
**Migration**: Use OpenSpec commands for creating specifications.
